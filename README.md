# Synth

A simple browser-based synthesizer using the Web Audio API.

## How to use

Open `index.html` in any modern browser — no install needed.

- **Click** keys on the keyboard to play notes
- **Keyboard shortcuts**: `A S D F G H J K L ;` for white keys, `W E T Y U O P` for black keys
- **Waveform**: switch between Sine, Square, Sawtooth, Triangle
- **Volume**: drag the slider
